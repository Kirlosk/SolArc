Technology Stack

Backend Framework

FastAPI powers our high-performance REST API, delivering predictions with minimal latency and automatic API documentation.

Machine Learning

XGBoost regression models trained on three specialized regions (Karwar, Hassan, Bengaluru) with climate-specific features for maximum accuracy.

Weather Integration

Open-Meteo API provides real-time POA irradiance, temperature, and wind speed data at high spatial resolution across 240+ cities.

Frontend Technologies

Modern HTML5, CSS3, and JavaScript with Anime.js animations, Chart.js visualizations, and responsive design for seamless user experience.

Solar Position Calculation

NOAA solar position algorithm computes real-time solar elevation angles for accurate irradiance predictions.

Data Processing

StandardScaler normalization, feature engineering with ground-reflected POA, and regional model assignment for optimal performance.

Model Architecture & Performance

SolArc employs XGBoost gradient-boosted regression trees optimized for renewable energy forecasting. Our system uses three region-specific models trained on historical weather data and solar production patterns:

Karwar Model (Coastal)

Trained for high-humidity coastal regions with monsoon patterns. Handles variable cloud cover and sea breeze effects on solar irradiance.

Hassan Model (Central)

Optimized for moderate climate zones with stable weather patterns. Captures agricultural region characteristics and seasonal variations.

Bengaluru Model (Urban)

Accounts for urban heat island effects, air pollution impacts, and metropolitan weather patterns affecting solar generation.

Each model uses six key features: POA ground-reflected irradiance, solar elevation angle, ambient temperature, wind speed, latitude, and longitude. StandardScaler normalization ensures consistent predictions across different magnitudes. The models predict irradiance (W/m¬≤), which is then converted to energy output (kWh) based on panel area and efficiency.
